# Description
Program ini bertujuan untuk membaca seluruh isi (text) file yang bertipe .txt

# HOW TO WORK
1. buka file "READ.txt"
2. tulis/paste text yang ingin dibaca
3. save file tersebut
4. kembali ke folder awal
5. klik "click here!!!"
